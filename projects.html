Projects

  • Smart Fruit Ripeness Detection and Automated Harvesting
    This project focuses on identifying the ripeness of fruits using an ESP32-CAM and basic machine learning techniques. The camera captures fruit images, processes them using edge computing, and determines whether the fruit is ripe or unripe. Based on the result, a robotic mechanism can automatically harvest the fruit. This system helps farmers reduce manual effort, save time, and improve harvesting accuracy in smart agriculture.

  • Smart Cattle Health Monitoring System
    Developed an IoT-based smart collar for cattle using ESP32, temperature sensors, accelerometer, and GPS module. The system continuously monitors the cattle’s body temperature, movement, and location. If any abnormal condition is detected, alerts are sent to the farmer. This helps in early disease detection, better livestock management, and reduced animal loss.
